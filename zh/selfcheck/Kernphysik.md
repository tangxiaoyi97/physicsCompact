## Zettel 1 — 放射性、核素图与半衰期(Radioaktivität, Nuklidkarte & Halbwertszeit)

> **范式定位(Paradigmeneinordnung):** 经典物理是一座钟表宇宙——知道现在,就能算出未来。而原子核的衰变第一次告诉我们:**自然自己也不知道**一个原子何时衰变。本卷从「核为何不稳定」走到衰变定律,最后走到这扇哲学之门。

1. **Erkläre:** 什么是放射性(Radioaktivität)?为什么它==与外界影响无关==(加热、加压都改变不了半衰期)?
2. **四个成因 + 核素图(Nuklidkarte):** 核不稳定的**四个原因**及核素图的**颜色编码**(黄/蓝/红/白/黑)。一个**蓝色**核素如何走回稳定谷?
3. **四种射线:** α、β⁻、β⁺、γ 各由什么组成?用**两条守恒律**(电荷、核子数)自己推出 $\Delta Z$、$\Delta A$,并配平:$^{238}_{\;92}\text{U}\rightarrow\ ?$
4. **Ableitung:** 从「每过 $T_{1/2}$ 剩一半」推出衰变定律 $N(t) = N_0\cdot2^{-t/T_{1/2}}$,再换底推出 $\lambda = \ln 2/T_{1/2}$。
5. **Rechenbeispiel(先写纸上,再讨论):** 切尔诺贝利的 **Cs-137**($T_{1/2} = 30{,}2$ a):事故 **36 年**后还剩多少?
   附加单位换算:I-131($T_{1/2}=8$ d)的衰变常数 $\lambda$ 是多少 **1/s**?
6. **科学理论/哲学弯钩:** ==衰变是真正的量子随机(echter Quantenzufall)==——衰变**有原因**,衰变**时刻却没有原因**。这对**拉普拉斯妖(Laplace'scher Dämon)**——「知道宇宙现状者可算出全部未来」——意味着什么?

## §§ANTWORTEN

### Zettel 1

1. 不稳定核**自发**衰变为另一个核并放出**电离辐射**,直到达到稳定同位素。==与外界无关==,因为核内能量在 **MeV** 量级(强核力),比化学能(eV,电子壳)大百万倍——外界的温度、压强对核「不可见」。德语关键句:„Der Kernzerfall ist unabhängig von äußeren Einflüssen — nichts ändert die Halbwertszeit."([[zh/themen/Kernphysik/12.3#12.3 放射性——成因与核素图|§12.3]])
2. **质量过大 → 黄(α);中子过多 → 蓝(β⁻);质子过多 → 红(β⁺);能量过多 → 白(γ);稳定 = 黑**。走法:β⁻ = $Z{+}1,N{-}1$ 左上一格;β⁺ = 右下一格;α = 左下斜两格;γ 原地。**蓝色核素**(谷的右下侧)经**一连串 β⁻ 衰变**一步步向左上爬,直到落进黑色稳定谷。([[zh/themen/Kernphysik/12.3#12.3 放射性——成因与核素图|§12.3]])
3. **α** = 氦核(2p+2n),$\Delta Z=-2,\Delta A=-4$;**β⁻** = 电子($n\to p^++e^-+\bar\nu_e$),$\Delta Z=+1,\Delta A=0$;**β⁺** = 正电子($p^+\to n+e^++\nu_e$),$\Delta Z=-1,\Delta A=0$;**γ** = 高能光子,$\Delta Z=\Delta A=0$。配平:$^{238}_{\;92}\text{U}\rightarrow{}^{234}_{\;90}\text{Th}+{}^{4}_{2}\text{He}$——电荷 $92 = 90+2$ ✓,核子 $238 = 234+4$ ✓。([[zh/themen/Kernphysik/12.4#12.4 四种射线|§12.4]])
4. 数折半次数:过 $n$ 个半衰期剩 $N_0\cdot2^{-n}$;$n = t/T_{1/2}$ → $N(t) = N_0\cdot2^{-t/T_{1/2}}$。换底:$2 = e^{\ln2}$ → $2^{-t/T_{1/2}} = e^{-(\ln2/T_{1/2})t}$,与 $e^{-\lambda t}$ 比较 → $\lambda = \ln2/T_{1/2}$。$\lambda$ = 单个核单位时间的衰变概率。([[zh/themen/Kernphysik/12.7#12.7 半衰期与衰变定律|§12.7]])
5. **计算(已验算):** $n = \dfrac{36}{30{,}2}\approx1{,}19$ 个半衰期;$2^{-1{,}19} = 1/2^{1{,}19}$,$2^{1{,}19}\approx2{,}28$ → $1/2{,}28\approx0{,}438 = \mathbf{43{,}8\,\%}$。**验算:** $1{,}19\cdot\ln2 = 1{,}19\cdot0{,}693\approx0{,}826$,$e^{-0{,}826}\approx0{,}438$ ✓——36 年后仍剩近一半,这正是数十年级半衰期「棘手」之处。**单位换算:** $T_{1/2} = 8\ \text{d} = 8\cdot86\,400\ \text{s} = 691\,200\ \text{s}$;$\lambda = \dfrac{\ln2}{691\,200\ \text{s}}\approx\dfrac{0{,}693}{6{,}912\cdot10^5\ \text{s}}\approx1{,}0\cdot10^{-6}\ \tfrac{1}{\text{s}}$。**验算:** $1{,}0\cdot10^{-6}\cdot6{,}912\cdot10^5\approx0{,}69\approx\ln2$ ✓。([[zh/themen/Kernphysik/12.7#12.7 半衰期与衰变定律|§12.7]])
6. 拉普拉斯妖的前提是**决定论**:同样的现在 → 同样的未来。量子随机打破它:衰变时刻==**没有**原因==,即使知道核的全部状态也算不出衰变时刻——„Die Natur selbst weiß nicht, wann ein Atom zerfällt."妖**必败**:这不是信息不足(人类无知),而是**自然本性**;能预言的只剩**概率**(半衰期正是统计陈述)。([[zh/themen/Kernphysik/12.7#12.7 半衰期与衰变定律|§12.7]])

### Zettel 2

1. C-14 在**大气**中由**宇宙射线**生成:中子击中 N-14:$^{14}_{7}\text{N}+{}^{1}_{0}n\rightarrow{}^{14}_{6}\text{C}+{}^{1}_{1}\text{H}$。配平:电荷 $7+0 = 6+1$ ✓;核子 $14+1 = 14+1$ ✓。([[zh/themen/Kernphysik/12.8#12.8 C-14 测年法（放射性碳）|§12.8]])
2. β⁻ 衰变回 N-14:$^{14}_{6}\text{C}\rightarrow{}^{14}_{7}\text{N}+e^-+\bar\nu_e$。检查:电荷左 $6$,右 $7+(-1) = 6$ ✓;核子 $14 = 14$ ✓。生成与衰变构成**循环**。
3. 活体不断与环境**交换碳**(呼吸、进食)→ 体内比例 = 大气比例(大气中不断新生 C-14,长期近似恒定);**死亡后停止摄入**,C-14 只衰变不补充 → 比例按 $2^{-t/T_{1/2}}$ 下降。约 **9 个半衰期**后只剩 $2^{-9}\approx0{,}2\,\%$,测不动了 → 上限约 5 万年;更老用钾-氩法。([[zh/themen/Kernphysik/12.8#12.8 C-14 测年法（放射性碳）|§12.8]])
4. **计算(已验算):** $12{,}5\,\% = \dfrac{1}{8} = 2^{-3}$ → 恰好 **3 个半衰期** → $t = 3\cdot5730\ \text{a} = \mathbf{17\,190}$ **年**。**验算:** $2^{-3} = 1/8 = 0{,}125 = 12{,}5\,\%$ ✓;$3\cdot5730 = 17\,190$ ✓。
5. **β⁺ 放射体**放出**正电子** → 与电子**湮灭**:$e^++e^-\to\gamma+\gamma$。**两个 γ、180°:** 湮灭前总动量 ≈ 0,单个光子动量 ≠ 0 → 必须两个**等大反向** → 180°。**符合:** 两个相对探测器**同时**击中 → 湮灭在**连线**上。**4 个 γ:** 一次符合只定一条**线**,两条线的**交点**才是点 → 2 次湮灭 = 4 γ。德语关键句:„Eine einzelne Koinzidenz legt nur eine Linie fest, erst der Schnittpunkt zweier Linien einen Punkt."([[zh/themen/Kernphysik/12.9#12.9 正电子发射断层成像（PET）|§12.9]])
6. **计算(已验算):** $E = m_ec^2 = 9{,}11\cdot10^{-31}\ \text{kg}\cdot(3\cdot10^8\ \tfrac{\text{m}}{\text{s}})^2 = 9{,}11\cdot10^{-31}\cdot9\cdot10^{16}\ \text{J}\approx8{,}2\cdot10^{-14}$ J。换算:$\dfrac{8{,}2\cdot10^{-14}}{1{,}6\cdot10^{-19}}\ \text{eV}\approx5{,}1\cdot10^{5}\ \text{eV}\approx\mathbf{511\ keV}$。**验算:** $9{,}11\cdot9 = 82{,}0$ → $8{,}2\cdot10^{-14}$ ✓;$8{,}2/1{,}6 = 5{,}125$ ✓。
7. **F-18:** $T_{1/2}\approx110$ min,够从回旋加速器**运送**到医院,并嵌入葡萄糖类似物 **FDG**(肿瘤/活跃脑区耗糖多 → 富集发亮)。**O-15** 只有 $T_{1/2}\approx2{,}03$ min:晚到 13 min = $n\approx13/2{,}03\approx6{,}4$ 个半衰期 → 剩 $2^{-6{,}4}\approx1{,}18\,\%$——几乎全没了(所以科研负责人恼火)。**验算:** $2^{6{,}4}\approx64\cdot1{,}32\approx84{,}5$,$1/84{,}5\approx0{,}0118$ ✓。([[zh/themen/Kernphysik/12.9#12.9 正电子发射断层成像（PET）|§12.9]])
8. PET 是「纯理论」变日常的范例:狄拉克预言的**反物质**(1928)、爱因斯坦的 $E=mc^2$(1905)——当年毫无「用处」,今天每天在肿瘤科救命。结论:基础研究的应用**不可预订**,边界只是时间差。

### Zettel 3

1. 电离辐射从分子中**打出电子**;若击中 **DNA**,细胞「图纸」受损,修复出错可致**失控分裂(癌)**。**频繁分裂的细胞**(骨髓、黏膜、胚胎)尤其敏感——它们常「使用」DNA,错误传播快。([[zh/themen/Kernphysik/12.6#12.6 辐射防护|§12.6]])
2. **Aufenthaltszeit(停留时间)短 · Abstand(距离)大 · Abschirmung(屏蔽)**(铅、铅玻璃、钢筋混凝土)。射程:**α** 一张纸/皮肤;**β** 几 mm 铝;**γ** 厚铅/混凝土(穿透力最强)。([[zh/themen/Kernphysik/12.6#12.6 辐射防护|§12.6]])
3. 点源向各方向均匀辐射 → 辐射摊在**球面** $O = 4\pi r^2$ 上 → 面积随 $r^2$ 增长 → 单位面积强度 $I\propto1/r^2$。与引力、库仑定律同根:**三维空间里点源的几何**。([[zh/themen/Kernphysik/12.6#12.6 辐射防护|§12.6]])
4. **计算(已验算):** $\dfrac{I_2}{I_1} = \left(\dfrac{r_1}{r_2}\right)^2 = \left(\dfrac{0{,}5}{2}\right)^2 = \left(\dfrac{1}{4}\right)^2 = \dfrac{1}{16} = 6{,}25\,\%$。**验算:** 距离 ×4 → 强度 ÷ $4^2 = 16$ ✓——后退几步是最便宜的防护。
5. 金属管内低压气体;**外壁(−)与中心丝(+)之间加高压**(强电场)。入射粒子**电离**气体原子 → 自由电子被场**加速**、沿途再电离 → **雪崩** → 电荷到达电极 = 一次**电流脉冲**(「咔哒」)。**间接**:看不见粒子本身,数的是它的**电离效应**。德语关键句:„Man zählt Ereignisse, nicht die Strahlung selbst."([[zh/themen/Kernphysik/12.5#12.5 盖革-米勒计数管|§12.5]])
6. 局限:**区分不了** α/β/γ、**测不了能量**;不带电的 **γ** 必须先在管壁/气体中打出电子才能触发 → 探测效率显著更低。([[zh/themen/Kernphysik/12.5#12.5 盖革-米勒计数管|§12.5]])
7. 太空中既无大气(≈ 10 m 水的等效屏蔽)也无地磁场 → 火星任务用 **~5 m 厚的水层包住飞船**(水富氢:慢化 + 吸收,且反正要带)、**磁屏蔽**(模仿地磁场)、**地下舱室**(见[[zh/themen/Physikalische Naturphaenomene/6.1#6.1 宇宙射线|主题 6 §6.1]])。宇航员问题大得多:剂量率更高(无任何屏蔽)且任务连续数年;飞行常旅客只在 ~10 km 高度、仍有大部分大气保护。

### 追问链

1. **离子**变的是**壳**(电子数),**同位素**变的是**核**(中子数)。
2. $^4_2$He 核**结合得格外牢**(每核子结合能高)——整包甩掉能量上最「划算」。
3. 一个**下夸克 → 上夸克**(udd → uud),经**弱力**,放出 **$W^-$ 玻色子**,它衰变为 $e^-+\bar\nu_e$。
4. 不是为电荷(它中性),而是为**轻子数守恒**;并分走能量 → β 电子连续能谱。
5. 自由的 $p\to n$ 要**吸能**(质子更轻)→ 稳定;核内由**整个核的结合能之差**支付 → 可行。
6. 库仑斥力长程($\sim Z^2$)、强核力短程 → 大核需要「只供胶水不添斥力」的额外中子;**铅($Z=82$)之后没有稳定核素**。
7. 没变——γ 只带走能量,$Z$、$A$ 都不变。
8. 只对**大量**核才有意义:单个核的衰变时刻完全随机,只有系综遵守 $2^{-t/T_{1/2}}$。
9. **不变**——核内能量(MeV)比热能(eV 以下)大百万倍,外界对核「不可见」。
10. **β⁻** 左上一格($Z{+}1,N{-}1$);**α** 左下斜两格($Z{-}2,N{-}2$)。
11. $6500$ 万年 ≈ $11\,000$ 多个半衰期 → $2^{-11000}$,什么都不剩;9 个半衰期(约 5 万年)就到测量极限。化石要用钾-氩法($T_{1/2}\approx12{,}5$ 亿年)。
12. 各 **511 keV** = $m_ec^2$——电子静止质量的能量当量(见 Zettel 2 第 6 题计算)。
