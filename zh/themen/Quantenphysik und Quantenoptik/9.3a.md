# 9.3a 薛定谔方程、德布罗意物质波与玻恩诠释

> **来源：** Mappe S. 95 · ➕ 考纲

§9.3 引入了波函数 $\psi$，并指出 $|\psi|^2$ 是出现概率。还有两个问题悬而未决：**$\psi$ 从何而来**（由哪条方程决定？）以及**为什么**连**物质**也有波动性？这由德布罗意、薛定谔和玻恩来回答。

## 德布罗意物质波（de-Broglie-Welle，1924）

**路易·德布罗意（Louis de Broglie）**把波粒思想反转过来：既然**光**（波）也是粒子（光子），那么**物质**（粒子）也应是**波**。给每个动量为 $p = m v$ 的粒子配一列==**物质波（德布罗意波）**==，其波长为：

$$\boxed{\lambda = \frac{h}{p} = \frac{h}{m\,v}}$$

| 符号 | 含义 | 单位 |
| :-- | :-- | :-- |
| $\lambda$ | **德布罗意波长（de-Broglie-Wellenlänge）** | m |
| $h$ | 普朗克常数 $= 6{,}626\cdot10^{-34}$ | Js |
| $m$ | 粒子质量 | kg |
| $v$ | 粒子速度 | m/s |

✎ **为什么日常感觉不到？** 对**宏观**物体而言 $\lambda = h/(mv)$ 小得离谱（飞行的台球 $\lambda \sim 10^{-34}$ m——无法测量）。只有在**质量极小**（电子）时 $\lambda$ 才大到足以显示衍射/干涉——这正是[[zh/themen/Quantenphysik und Quantenoptik/9.3#9.3 双缝实验|双缝]]中电子所发生的事。

## 戴维森-革末实验（Davisson-Germer，1927）——实验证明

==**戴维森与革末**把电子射向一块**镍晶体（Nickel-Kristall）**，观察到**衍射图样**==——这本是只有波在光栅上才会产生的（规则的晶格起到衍射光栅的作用，参见[[zh/themen/Optik und Wellenphaenomene/|光学]]）。==由此德布罗意的物质波得到**实验证明**：==电子（经典粒子！）显示出**衍射**，且测得的波长与 $\lambda = h/(mv)$ 吻合。**电子衍射**如今是**电子显微镜**的基础（$\lambda$ 短 → 分辨率高）。

## 薛定谔方程（Schrödinger-Gleichung，1926）

**埃尔温·薛定谔（Erwin Schrödinger）**于 1926 年提出**量子力学的基本方程**——它对波函数 $\psi$ 所起的作用，就如牛顿的 $F = ma$ 对经典轨道的作用：==它决定 **$\psi$ 如何随时间演化**。==其含时形式为：

$$\boxed{i\,\hbar\,\frac{\partial \psi}{\partial t} = \hat{H}\,\psi}$$

| 符号 | 含义 |
| :-- | :-- |
| $\psi$ | 波函数（复数；$\psi = \psi(\vec r, t)$） |
| $\hat{H}$ | **哈密顿算符（Hamilton-Operator）**（代表总能量：动能 + 势能） |
| $\hbar$ | 约化普朗克常数 $= h/2\pi$ |
| $i$ | 虚数单位（$i^2 = -1$） |
| $\frac{\partial \psi}{\partial t}$ | $\psi$ 随时间的变化 |

✎ **这条方程直观上是什么意思？** **哈密顿算符 $\hat H$** 编码了作用在粒子上的一切力/能量（例如原子核的吸引）。方程随即**确定性地**预言 $\psi$ 如何随时间改变。要点：==薛定谔给出的是 **$\psi$ 的演化**——而 $\psi$ *意味着什么*（即 $|\psi|^2$ = 概率，§9.3）是一条**独立**的论断（玻恩，见下文）。==把这条方程用于一个束缚电子求解，就能得出原子的离散能级（§9.2、§9.2a），**完全不需要**玻尔那条人为的轨道假设。

✎ 凭此方程，**薛定谔获得 1933 年诺贝尔奖**（与狄拉克 Dirac 共享）。

## 玻恩诠释（Born'sche Deutung，马克斯·玻恩 1926）

薛定谔本人起初以为 $\psi$ 是一列真实的波。==**马克斯·玻恩（Max Born）**于 1926 年给出至今通行的**概率诠释**：$|\psi|^2$ 是**概率密度（Wahrscheinlichkeitsdichte）**==——在某点附近小体积 $\mathrm{d}V$ 内找到粒子的概率为 $|\psi|^2\,\mathrm{d}V$。（这是 §9.3 中「$|\psi|^2$ = 出现概率」的数学精确形式。）

由此得出==**归一化条件（Normierung）**==：既然粒子**必定**在空间**某处**，那么对整个空间的总概率等于 $1$：

$$\boxed{\int |\psi|^2 \, \mathrm{d}V = 1}$$

✎ **归一化的意义：**它是「100% 概率」的量子版本。它约束了薛定谔方程的允许解 $\psi$（只有**可归一化**的 $\psi$ 才有物理意义）——而正是这条边界条件，迫使束缚粒子出现**离散**能级（见 §9.3b 势阱）。==凭此诠释，**玻恩获得 1954 年诺贝尔奖**。==有了玻恩，量子力学的**非决定论**最终确立：$\psi$ 虽确定性地演化（薛定谔），**测量结果**却只能**按概率**预言——这是量子随机性（§9.8）的根源。
