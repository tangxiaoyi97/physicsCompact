# 13.9 声学：乐音、噪声、响度

> **来源：** ➕ 考纲 · 承接 §13.4（纵波）

==**声音（Schall）**是一种**纵波**：介质中一连串的**压强起伏**（压缩/稀疏）==（空气、水、固体）。它与光不同，**始终需要介质**（真空中：寂静）。空气中的**声速**约为

$$\boxed{c_\text{声} \approx 343\ \text{m/s}}\quad(\text{空气，}20\,°\text{C})$$

——比光约慢**一百万倍**（所以先见闪电**后**闻雷）。这里同样有 $c_\text{声}=\lambda f$。

## 乐音、复音、噪声——三种声型

| 概念 | 振动形式 | 例子 |
| :-- | :-- | :-- |
| **乐音 / 纯音（Ton）** | **纯正弦振动**——**单一**频率 | 音叉、正弦信号发生器 |
| **复音（Klang）** | **周期性**：基音 **+ 泛音** | 乐器、歌声（元音） |
| **噪声（Geräusch）** | **非周期**——无固定音高的**频率混合** | 白噪声、爆响、嘶声 |

> ✎ **音高 ↔ 频率，响度 ↔ 振幅。** 一段声振动有两个独立属性：
> - ==**音高（Tonhöhe）**由**频率** $f$ 决定==——频率高 = 音高高（人耳约听 **20 Hz – 20 kHz**）。
> - ==**响度（Lautstärke）**取决于**振幅** $A$==——振幅大 = 响（参光：亮度同样在 $A$ 里，§13.6）。

## 泛音与音色

> ✎ **同一个「啊」，小提琴和钢琴为何听起来不同？（口试常问）** 真实的**复音（Klang）**绝非纯正弦，而是除**基音（Grundton）**（频率 $f_1$，决定音高）外，还含整数倍 $2f_1, 3f_1, \dots$——即**泛音 / 谐波（Obertöne, Harmonische）**。==**音色（Klangfarbe）**取决于**各泛音的相对强弱**。==小提琴和钢琴奏同一基音（音高相同），但**泛音构成不同** → **音色**不同。数学上这是一次**傅里叶分解**：任何周期振动都可写成正弦音之和。

## 响度与分贝（dB）

> ✎ **为何用*对数*标度？（声压级）** 人耳覆盖**极大**的强度范围（听阈到痛阈约 $10^{12}$ 倍），且==对响度的感知是**对数**的==（强度的相同*倍数*被听成相同*步长*）。所以用**分贝（Dezibel, dB）**这一对数标度表示**声压级（Schallpegel）**：
> $$\boxed{L = 10\cdot\log_{10}\!\frac{I}{I_0}\ \text{dB}}\,, \qquad I_0 = 10^{-12}\ \tfrac{\text{W}}{\text{m}^2}\ (\text{听阈})$$
> 经验法则：==**0 dB** = 听阈，约 **120 dB** = 痛阈。==强度增**十倍**为 **+10 dB**；增**一倍**约 **+3 dB**。

| 声压级 | 例子 |
| :-- | :-- |
| 0 dB | 听阈 |
| ~30 dB | 耳语 |
| ~60 dB | 正常交谈 |
| ~120 dB | 痛阈（近处喷气机） |

> ✎ **距离律——越远越轻。** 点状声源的声功率分布到越来越大的球面上 → **声强**按 ==$I\propto \dfrac{1}{r^2}$== 衰减（与辐射普遍的 $1/r^2$ 律相同，见[[zh/themen/Physikalische Naturphaenomene/|主题 6]]）。距离加倍，$I$ 降到四分之一（$-6$ dB）。

> ✎ **衔接：主动降噪。** 降噪基于两列声波的**相消干涉**（反相、错开 $\pi$）——详见 §13.5。它利用的正是支撑此处声学的同一套波动物理。
