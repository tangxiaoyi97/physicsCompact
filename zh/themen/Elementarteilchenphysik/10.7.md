# 10.7 数量级阶梯（概览）

> **来源：** ✎ 补充（数量级概览）

为了理解基本粒子物理究竟深入到**多小**，可借助一架**数量级的阶梯**——从日常世界一路向下到已知最小的组成单元。每一级大约比上一级小一个数量级（甚至更多）：

| 数量级 | 对象 |
| :-- | :-- |
| $\sim 10^{0}\text{–}10^{1}$ m | **人体**（日常尺度） |
| $\sim 10^{-3}$ m | **沙粒 / 细胞团** |
| $\sim 10^{-5}$ m | **细胞** |
| $\sim 10^{-9}$ m | **分子 / DNA 直径**（纳米） |
| $\sim 10^{-10}$ m | **原子**（埃 Ångström） |
| $\sim 10^{-14}\text{–}10^{-15}$ m | **原子核**、**核子**（质子/中子） |
| $< 10^{-18}$ m | **夸克 / 电子**——点状，无可测量的大小 |

==于是粒子物理的跨度约达 **19 个数量级**：从人体（$10^{1}$ m）一直到夸克与电子（$<10^{-18}$ m）。== ✎ 在标准模型中，夸克与轻子被视为==**点状**==（无内部结构）——$<10^{-18}$ m 只是实验给出的**上限**：在此分辨率以内**未发现**任何大小。正是为了分辨更**小**的尺度，按 $E=mc^2$ 就需要越来越**高**的能量——这也是为什么要建越来越大的加速器，如 LHC（§10.5）。
