# 10.5 CERN 的实验粒子研究

> **来源：** [[Themenpool_Physik_Matura_2026#12) Kernphysik|考纲 §12]] · 课堂笔记 §2.3

**CERN**（位于日内瓦附近，✎ 1954 年作为欧洲战后**和平合作项目**创立）运行着世界上最大的粒子加速器——**LHC（大型强子对撞机）**（✎ 环形隧道周长约 **27 km**，位于地下约 100 m；「强子」之名见 §10.2——对撞的质子/铅核都是强子）。

**结构与工作方式（逐步）：**

1. **为什么要加速器？** 按 $E=mc^2$，**产生重粒子**（希格斯、$W/Z$、新粒子）需要**很多能量**。这能量以**动能**的形式注入质子：碰撞时质子越快（能量越高），撞击时就能「由能量」产生越重的粒子。
2. **加速过程：**质子先在**直线加速器**中预加速，再注入**同步加速器**（环形）中跑许多圈，每圈获得能量。这里有两件事是**分开**完成的：
   - **维持圆轨道——洛伦兹力。**真正把带电质子约束在环形轨道上的力是==**洛伦兹力（Lorentzkraft）**== $\vec F = q\,\vec v \times \vec B$：磁场 $\vec B$ 使每个运动的带电粒子沿垂直于其速度的方向偏转，把轨道弯成圆。LHC 的偏转磁体为此产生极强的磁场，约 ==**8 特斯拉（Tesla）**==——这只有用==**超导**==磁体才能实现（无损耗、超大电流；见[[zh/themen/Elektrodynamik/14.8#14.8 超导|主题 14 §14.8]]，应用 CERN/LHC）。
   - **加速——腔（Kavität）。**让质子变快的不是磁体，而是==**腔（Kavität / 谐振腔，英文 cavity）**==：内有高频**交变电场**的空腔，它精确地在恰当节拍上「推」一把，使质子每经过一次就获得一份能量增益（即作用在运动电荷上的加速电场）。
3. **为什么用环（对撞机）？** 两束质子**反向**运行并被引导**正面对撞**——这样两者能量相加（远比打一个静止靶有效）。
4. **四个探测器**位于交叉点，记录碰撞中**产生了哪些粒子**、飞向何方：**ATLAS** 与 **CMS**（通用型，含希格斯搜寻）、**ALICE**（夸克-胶子等离子体）、**LHCb**（物质-反物质不对称）。

## 研究目标

**1) 暗物质**——解释为什么**星系边缘的恒星**转得比由可见质量算出的更快；有些星系团需要多达 **400 倍**的质量才能维系。它通过引力相互作用，但其余不可见——目前仍是理论构想。

**2) 暗能量**——解释宇宙**越来越快的膨胀**；可见物质只占总能量极小一部分。

**3) 希格斯理论 / 希格斯粒子**——解释==**粒子的质量从何而来**==：

- **希格斯场**充满**整个**空间（它无处不在，连「空」的真空里也有）。
- ✎ **机制（形象地）：**一个穿过此场运动的粒子，会被场「拖拽」，仿佛要在**黏稠糖浆**里挤行——这种对加速的阻力*正是*我们感受到的**（惯性）[[zh/themen/Klassische Mechanik/8.1#8.1 质量、密度、质心|质量]]**。
- ==粒子与希格斯场作用**越强**，其质量**越大**。==**顶夸克**耦合强 → 极重；**光子**则**完全不**耦合 → **无质量**（因而以光速飞行）。
- **希格斯粒子**是这个场的「激发」（如糖浆里的一道波）。它的探测证实了此场确实存在。==**2012 年在 CERN（ATLAS 与 CMS）被证实。**==

**4) 夸克-胶子等离子体**——夸克**自由**（不被束缚）的状态。极难制造（需巨大能量），但很重要，因为夸克因**禁闭**（指夸克被「囚禁」：强力随距离增大反而更强，故夸克永远无法单独挣脱——强行拉开时，能量宁可生成一对新夸克）否则无法单独研究。假说：**夸克星**（介于中子星和黑洞之间，靠**泡利原理**稳定）——尚未发现。✎ 夸克-胶子等离子体正是==**大爆炸后最初几微秒**==整个宇宙所处的状态——**ALICE** 探测器用**铅核（重离子）对撞**把它短暂重现，相当于在实验室里回看宇宙的「第一微秒」。

**5) 超对称（SUSY）**——假设每个玻色子有一个超对称费米子，反之亦然（**玻色子** = 整数自旋、力的传递者；**费米子** = 半整数自旋、物质）。候选 **WIMP**（弱相互作用大质量粒子）——只参与弱力 + 引力、质量大；最轻者为 **中性伴子（Neutralino）**（暗物质的可能成分）。至今未发现 WIMP → 超对称已不再被视为确定。

**6) 微型黑洞**——弦论（一种理论框架，把所有基本粒子不看作点，而看作极微小的振动「弦」（即 Strings），并试图借此把引力也纳入其中）的检验，比质子还小；寿命约 $10^{-26}$ s → ==**完全无害**==（经[[zh/themen/Gravitationsphysik/11.7#11.7 黑洞|霍金辐射]]立即蒸发）。

**7) 物质-反物质不对称**——**论证**逐步如下：

1. 大爆炸时，按 $E=mc^2$，能量应**成对**产生粒子与反粒子（即两者**等量**）。
2. 若此后等量相遇，它们会**完全湮灭**（§10.3）——只剩**辐射**，**没有**物质。那将没有恒星、没有地球、**没有「我们」**。
3. 我们的存在本身证明：必定曾有一份**微小的物质过剩**。计算出的初始比例是

$$\text{粒子} : \text{反粒子} = (10^9 + 1) : 10^9$$

   即**每十亿个反粒子，就对应十亿零一个粒子**。
4. 那十亿对粒子-反粒子湮灭为辐射；微小的**剩余**（每十亿中多出的那一个）构成了**今天所有**可见物质。

为何会有这份过剩（自然为何极轻微地偏爱物质），是现代物理的==**开放问题**==——CERN 的 **LHCb** 探测器正研究此事。

✎ 附带一提一个**假想**粒子：==**快子（Tachyon）**会是一种**超光速**粒子==（$v>c$）。若它存在，就会违反**因果律**（结果可能先于原因出现）——它**从未被发现**，也**不属于**标准模型。
