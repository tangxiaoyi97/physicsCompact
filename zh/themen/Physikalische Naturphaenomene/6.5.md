# 6.5 薄膜干涉

> **来源：** [[Themenpool_Physik_Matura_2026#6) Physikalische Naturphänomene|考纲 §6]]（另见[[zh/themen/Optik und Wellenphaenomene/13.5#13.5 干涉与衍射|光学]]）

## 现象与机制

肥皂泡、水面油膜、增透膜（眼镜、镜头）、蝴蝶翅膀、CD 上的**虹彩**。

机制：光一部分在薄膜**上表面**反射，一部分进入薄膜、在**下表面**反射后再射出。==两束反射光走过的路有一段**错位**，叠加在一起——发生**干涉**。==某处是亮还是暗，取决于两束波到达时是**同步**（峰对峰）还是**反相**（峰对谷）。

## 条件

这段「错位」用**相位差** $\Delta\varphi$ 来度量。$\Delta\varphi\dots$ 两束反射光的相位差；$n=0,1,2,\dots$ 级数（✎ 别与下文的**折射率** $n$ 混淆！）。

| 干涉 | 条件 | 图像 |
| :-- | :-- | :-- |
| **相长** | $\Delta\varphi = 2\pi n$ | 峰遇峰 → 亮 |
| **相消** | $\Delta\varphi = (2n+1)\pi$ | 峰遇谷 → 暗 |

✎ 一个完整波长 $\lambda$ 对应相位 $2\pi$。**相长**即：错位是 $\lambda$ 的**整数倍**（$\Delta\varphi=2\pi n$），峰与峰重合。**相消**即：错位是**半整数**倍（$\Delta\varphi=\pi,3\pi,\dots$），峰遇谷而相互抵消。

## 相位跃变与光程差

$\Delta\varphi$ 由两件事共同造成：(1) 下面那束光穿过薄膜的**几何绕行**，(2) 反射时可能出现的**相位跃变**。

**(1) 光程差 $2nd$ 从何而来？** 逐步看（**垂直**入射时）：
1. 下面那束光要在薄膜里**下去再上来**——几何上多走 $d$（下）$+\,d$（上）$=2d$。$d\dots$ 膜厚。
2. **但光在膜里更慢**：波长缩短为 $\lambda_\text{介质}=\dfrac{\lambda}{n}$。$n\dots$ 薄膜折射率。干涉看的不是**几何**路程，而是**光程**——即几何路程乘以 $n$（折算成里面装得下多少个真空波长）。
3. **光程差：** 几何绕行 $\times\,n$：$\;\boxed{\Delta s_\text{opt}=2nd}$。（斜入射时还要乘一个角度因子；垂直入射恰为 $2nd$。）

**(2) 相位跃变。** ==在**光密**介质（$n$ 更大）处反射，相位会**额外**跃变 $\pi$==（＝半个 $\lambda$）；在光疏介质处则不跃变。例如肥皂泡（空气 → 水 → 空气）：**上表面**（空气 → 水，进入光密）有跃变，**下表面**（水 → 空气，进入光疏）**无**跃变。==这一个相位跃变把条件反了过来==——所以极薄的肥皂膜（$d\to0$，即 $\Delta s_\text{opt}\to0$）在反射光中是**暗**而非亮：仅凭这一个相位跃变就已造成相消。

相位跃变遵循一条**波动学定理**（对任何波都成立，绳波也一样）：

> ==在**固定端**，波峰被反射为**波谷**（相位跃变 $\pi$）。==
> ==在**自由端**，波峰被反射为**波峰**（无跃变）。==

对光而言：在「空气 → 玻璃」界面（光密＝「固定端」）反射跃变 $\pi$；「玻璃 → 空气」界面（光疏＝「自由端」）不跃变。

> ✎ **算一个应用——λ/4 增透膜。** 玻璃（$n\approx1{,}5$）上镀一层折射率**居中**的薄膜（如 MgF₂，$n\approx1{,}38$）：因 $n_\text{空气}<n_\text{膜}<n_\text{玻璃}$，**两个界面都是向光密介质**反射 → **两束**反射波都跃变 $\pi$ → ==相位跃变相互抵消==，只剩光程差 $2nd$ 起作用。要让两束反射光**相消**，需 $2nd=\frac{\lambda}{2}$，即 ==$d=\dfrac{\lambda}{4n}$==——**四分之一波长**膜（按人眼最敏感的绿光 $\lambda\approx550$ nm 设计：$d\approx100$ nm）。*被消掉的能量去哪了？* 它并未消失，而是==进入了**透射**光==——这正是镀膜的目的（让更多光进入相机/眼睛/太阳能电池）。由于条件只对**一个**波长严格成立，残余反光呈**紫红/紫色**（红、蓝成分仍在）。

## 牛顿环（Newton'sches Farbenglas）

> **来源：** Mappe（Friedl）第 30–31 页

把一块微凸的**会聚透镜**放在平玻璃板上，两者之间形成一层向外渐厚的薄**空气层**。用**单色光**（只含单一波长的纯色光，例如激光）垂直照射，会看到**同心的明暗环**（反射光中央**暗**，透射光中央**亮**——两幅图像**互补**）。==用**蓝光**时环比红光**更密**==→ 蓝光**波长更短**。

明暗取决于空气层的**厚度 $d$**（相位跃变只发生在下界面「空气 → 玻璃」）：

$$\text{反射光暗纹：} 2d = 2k\cdot\frac{\lambda}{2} \qquad\qquad \text{透射光暗纹：} 2d = (2k+1)\frac{\lambda}{2} \qquad (k=0,1,2,\dots)$$

**环半径。** 设透镜曲率半径为 $R$、环半径为 $r$，由射影定理（圆几何中的一条定理：直角三角形中，斜边上的高的平方等于它在斜边上分出的两条线段之积）$r^2 = d(2R-d) \approx 2Rd$（因 $d \ll R$），即 $d = r^2/(2R)$。对反射光中第 $k$ 个暗环，$2\cdot\tfrac{r^2}{2R} = 2k\cdot\tfrac{\lambda}{2}$，于是：

$$\boxed{\lambda = \frac{r^2}{k\cdot R}}$$

==已知 $R$，量出环半径即可得到与光栅光谱（用光栅——许多细密、彼此紧靠的狭缝——经衍射测出的波长）**相同**的波长==——一个独立的验证。

> ✎ **反过来——精密测长。** 已知 $\lambda$，便可测量极小长度。两块玻璃板一边相接、另一边夹一根**头发**，形成空气劈；用红光（$\lambda = 650$ nm）照射出现暗纹。若第 41 条纹在头发处（$k=40$），则头发厚度 $d = 40\cdot\tfrac{650\cdot10^{-9}}{2}\,\text{m} = 13\cdot10^{-6}$ m。==1960 年国际规定：1 米 = 氪-86 橙红色谱线的 **1 650 763,73** 个波长==——一个可复现的长度定义（直到 1983 年改用光速定义）。

## 颜色依赖与应用

不同**波长**满足条件的程度不同 → 有的颜色相长、有的相消 → ==**虹彩**==。

✎ **为什么颜色随视角变化？** 斜入射时**光程差**会变（$\Delta s = 2nd\cos\beta$，$\beta$ 为膜内折射角）：看得越斜，光程差越**小** → 相长条件由**更短**的波长满足 → ==转动肥皂泡/蝴蝶翅膀时颜色向**蓝**偏移==。正是这种**视角依赖**把结构色与颜料色区分开（颜料从任何角度看都一样）。

**应用：** 增透膜（眼镜、相机、太阳能电池）、激光器中的光学镀膜、自然界的**结构色**（蝴蝶、孔雀羽、甲虫）。
