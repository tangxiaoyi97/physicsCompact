# 8.7 圆周运动与向心力

> **来源：** 课堂笔记 §4.4 · ➕ 考纲（[[zh/themen/Raumfahrtphysik/|航天]]与[[zh/themen/Gravitationsphysik/|引力]]的基础概念）

==**向心力** $F_\text{zp}$ 是把物体**保持在圆（或椭圆）轨道上**的力。==它**始终指向**轨道圆心。

**圆周运动为什么需要力？** 逐步来看：

1. 速度是一个**矢量** $\vec v$——它有大小（快慢）**还有方向**。
2. 在圆轨道上，$\vec v$ 的**方向不断改变**（物体一直在拐弯），即便快慢不变。
3. ==$\vec v$ 的**任何**方向改变都已经是一种加速度==（加速度＝速度**矢量**的改变，不只是快慢的改变——参见 §8.3 / [[zh/themen/Bewegungen/5.1#加速度|运动]]）。
4. 由牛顿第二定律 $\vec F = m\vec a$，加速度必须由**力**产生。这个指向圆心的力就叫**向心力**。

==向心力**不是一种独立的自然力**，而是一个由现成的力来担当的**角色**==——对卫星是**引力**，对链球运动员是**绳子**，对原子中的电子是**库仑力**（带电粒子之间的电吸引/排斥力）。所以它是一个**上位概念**。

> ✎ **那离心力（Zentrifugalkraft）呢？** 它==**不是真实的力**，而是一个**假想力（Scheinkraft）**==——是随转动参考系一起转时的**惯性效应**。直线运动里叫「惯性」的东西，在转动里就叫「离心力」。（坐旋转木马时*感觉*被甩向外，其实身体只是想沿直线继续——是外壁把它往里推。）

> ✎ **第二种假想力：科里奥利力（Corioliskraft）。** 离心力连**静止**于转动系中的物体也会「甩」向外，而==**科里奥利力**只作用于转动参考系中**运动着**的物体——它使其轨迹**侧向偏转**。==它同样**不是真实的力**，而是一个**假想力**（惯性效应）。
>
> **旋转木马例子（逐步）：** 两个小孩在转动的**旋转木马（Karussell）**上面对面坐着，**笔直地**互相扔球。从外面（惯性系）看，球确实**直线**飞行——但球在途中，木马已经**继续转动**，接球者已不在原地。对*随木马转动*的小孩来说，球仿佛走了一条**弯曲**轨迹、被「拉向一旁」——这种表观偏转便归因于科里奥利力。
>
> ==**在自转地球上的后果：**==
> - **信风**及大尺度气流被偏转（北半球向**右**，南半球向**左**）。
> - **低压区/气旋**因此在北半球**逆**时针旋转、在南半球**顺**时针旋转。
> - [[zh/themen/Klassische Mechanik/8.3#第一定律——惯性定律|傅科摆]]**摆动平面的旋转**也可解释为科里奥利效应——归根结底都是同一个地球自转的体现。

**向心力的大小：**

$$\boxed{F_\text{zp} = \frac{m \cdot v^2}{r}}$$

| 符号 | 含义 | 单位 |
| :-- | :-- | :-- |
| $F_\text{zp}$ | 向心力（指向圆心） | N |
| $m$ | 绕行物体的质量 | kg |
| $v$ | 轨道速度（切向） | m/s |
| $r$ | 轨道半径 | m |

==由此引出一个在航天与引力中反复出现的基本套路：==当某个具体的力（如引力）充当向心力时，就令 **$F_\text{zp} = F_\text{（该力）}$**，即可解出轨道——卫星、地球同步轨道（卫星绕行速度与地球自转恰好同步、因而始终「停」在同一地点上空的轨道）、开普勒第三定律的公式都是这样来的。
