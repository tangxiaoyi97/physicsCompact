# 2.1 预备：什么是向心力？

> ✎ **补充**——这一小段预备贯穿本章三道计算题。没有它，公式看起来就像死记硬背。

**向心力** $F_\text{ZP}$ 是把物体**保持**在圆（或椭圆）轨道上的力。它**始终指向**轨道的圆心。

**圆周运动为什么需要力？** 把思路拆成小步：

1. 速度是一个**矢量** $\vec v$——它有大小**还有方向**。
2. 在圆轨道上，$\vec v$ 的**方向不断改变**（物体一直在拐弯），即便大小（快慢）保持不变。
3. $\vec v$ 的**任何**方向改变都已经是一种**加速度**（加速度＝速度*矢量*的改变，不只是快慢的改变）。
4. 由牛顿第二定律 $\vec F = m\vec a$，加速度必须由**力**来产生。这个指向圆心的力就叫**向心力**。

==向心力不是一种独立的自然力，而是一个由现成的力来担当的**角色**==——对卫星是**引力**，对链球运动员是**绳子**，对原子中的电子是**库仑力**（指带电粒子之间的电吸引力或排斥力——这里是带正电的原子核吸引带负电的电子）。

> ✎ **那离心力（Zentrifugalkraft）呢？** 它**不是真实的力**，而是一个**假想力（Scheinkraft）**——是随转动参考系一起转时的惯性效应。在直线运动中叫「惯性」的东西，在转动中就叫「离心力」。

**向心力的大小**（马上要用到）：

$$F_\text{ZP} = \frac{m \cdot v^2}{r}$$

| 符号 | 含义 | 单位 |
| :-- | :-- | :-- |
| $F_\text{ZP}$ | 向心力（指向圆心） | N |
| $m$ | 绕行物体的质量 | kg |
| $v$ | 轨道速度 | m/s |
| $r$ | 轨道半径 | m |
