# 12.1 原子结构、周期表、同位素、离子

> **来源：** 课堂笔记 §3.1–§3.2

> ✎ **本节讲什么？** 在算衰变之前，必须先知道核**由什么组成**、**用哪个数**来描述它。本节正是干这个：三个特征数（$Z$、$N$、$A$），并由此引出**同位素**与**离子**两个概念。

原子由一个极小而致密的**核**（质子 + 中子）和一层**电子壳**构成。原子通常是**中性**的，因为正质子与负电子在**数目上相互抵消**（质子数 = 电子数）。每个质子带 $+e$，每个电子带 $-e$——大小都是 $e=1{,}6\cdot10^{-19}$ C（[[zh/themen/Elektrodynamik/14.2#14.2 电荷与电荷守恒|基本电荷]]）；数目相同时总和恰为零。

周期表按几个特征数排列元素：

| 量 | 符号 | 含义 |
| :-- | :-- | :-- |
| **原子序数** | $Z$ | 核内**质子**数（中性原子时 = 电子数） |
| **中子数** | $N$ | 中子数（很少等于 $Z$！） |
| **核子数** | $A$ | $A = Z + N$（核子 = 质子或中子） |

> ✎ **写法（理解衰变的关键）：** 一个核素写作 $^{A}_{Z}\text{X}$——左**上**是核子数 $A$，左**下**是原子序数 $Z$。例如 $^{14}_{6}\text{C}$ 有 $Z=6$ 个质子、$N=A-Z=14-6=8$ 个中子。每次衰变中正是这两个数在变（或不变）——所以这套写法很值得记。

==**同位素（Isotope）**==是**同一元素、$Z$ 相同但 $N$ 不同**的原子。

- **为什么化学性质（几乎）相同？** 原子的化学性质由**电子壳**决定，而电子数（中性原子时）只取决于 $Z$。同位素 $Z$ 相同 → 电子壳相同 → 化学行为几乎一致。
- **为什么危险性却不同？** 核的**稳定性**取决于**$N/Z$ 之比**。$N$ 过高或过低会使核失去平衡 → 变得**有放射性**（见 §12.3）。例：$^{12}\text{C}$（6p、6n）稳定，$^{14}\text{C}$（6p、8n）会衰变。

==**离子（Ionen）**==是**电子数改变**的原子/分子（不是质子数改变！）：电子**过多**→**负**，电子**不足**→**正**。电离可由高能、高温、电荷、放射性或化学反应引起。==记住区别：**离子**改变的是**壳**（电子），**同位素**改变的是**核**（中子）——前者是壳的化学/物理，后者是核物理。==
