## §§ANTWORTEN
1. $Z$ = **质子数**（原子序数），$N$ = **中子数**，$A=Z+N$ = **核子数**。**同位素**：$Z$ 相同、$N$ 不同（同一元素）。**离子**：**电子数**改变的原子（多电子为负、缺电子为正）。(§12.1)
2. **质量**过大 → **黄**（α）；**中子**过多 → **蓝**（β⁻）；**质子**过多 → **红**（β⁺）；**能量**过多 → **白**（γ）。稳定 = **黑**。(§12.3)
3. **α** = 氦核，$\Delta Z=-2,\ \Delta A=-4$；**β⁻** = 电子，$\Delta Z=+1,\ \Delta A=0$；**β⁺** = 正电子，$\Delta Z=-1,\ \Delta A=0$；**γ** = 电磁辐射，$\Delta Z=\Delta A=0$。(§12.4)
4. 一个**下夸克**经**弱力**变成**上夸克**，放出一个 **$W^-$ 玻色子**，它再衰变为电子 + 反中微子。于是中子（udd）变成质子（uud）；**反中微子**与电荷无关（它中性），它保证的是**轻子数守恒**并带走一部分能量（→ β 电子的连续能谱）。(§12.4)
5. 辐射使管内气体**电离**；自由电子形成短暂**电流**（计数脉冲/「咔哒」）。由此**间接测出强度**。(§12.5)
6. **停留时间**短、**距离**大、**屏蔽**（铅/钢筋混凝土）。距离起平方作用，因为辐射散布在随 $r^2$ 增大的球面上 → $I\propto 1/r^2$。(§12.6)
7. $T_{1/2}$ 是**一半**核衰变所需时间。它是**统计**概念：衰变有原因，但**时刻没有原因**——纯属**量子随机性**（只在核很多时才有意义）。(§12.7)
8. $40\,\text{d}/8\,\text{d}=5$ 个半衰期 → $N/N_0=2^{-5}=\tfrac{1}{32}\approx \mathbf{3.13\%}$。(§12.7)
9. 活体的 $^{14}\text{C}/^{12}\text{C}$ **恒定**；死亡后 C-14 衰变（β⁻，$T_{1/2}\approx 5730$ 年）→ 由残余量得**年龄**。C-14 在**大气**中由 N-14 经**宇宙射线**（中子）生成。(§12.8)
10. **β⁺ 放射体**放出**正电子**，与电子**湮灭** → 两个 **511 keV 的 γ** 沿 **180°** 飞出（动量守恒）。环形探测器以**符合**（同时到达）测量 → 两者连线定位（毫米级）。(§12.9)
11. **β⁻**：$Z{+}1,N{-}1$ → 左上一格；**β⁺**：$Z{-}1,N{+}1$ → 右下一格；**α**：$Z{-}2,N{-}2$ → 左下斜两格；**γ** 原地不动。**蓝色**核素（中子过多，在谷的右下侧）经**一连串 β⁻ 衰变**向左上爬，直到进入黑色稳定谷。(§12.3)
12. 库仑斥力**长程**、作用于所有质子对（$\sim Z^2$），强核力**短程**、只及近邻——大核需要额外中子「只供吸引、不添斥力」；铅（$Z=82$）以上无稳定核素。自由质子比中子**轻**，$p\to n$ 吸能 → 自由质子稳定；核内由**整个核**的结合能之差支付能量，β⁺ 才成为可能。(§12.3/§12.4)
13. O-15 的 $T_{1/2}\approx2$ min → 必须在回旋加速器旁现场制备、立即使用；**F-18**（$T_{1/2}\approx110$ min）足够长，可**运输**并嵌入葡萄糖类似物 **FDG**，在肿瘤/活跃脑区富集成像。(§12.9)
14. **汤姆逊原子模型**（1904，**葡萄干布丁模型**）中**正电荷均匀**分布于球体，**电子**像**蛋糕里的葡萄干**一样嵌在其中——**没有原子核**。卢瑟福推翻了它：均匀「抹开」的正电荷绝不可能把 α 粒子**弹回**——观测到的反弹要求一个**又小又致密的核**。(§12.2)
15. **衰变系**是一连串衰变的**链条**，直到抵达**稳定**终核。从 **U-238 → Pb-206**：$\Delta A=32 \Rightarrow$ **8 次 α 衰变**；它们使 $Z$ 降 16（到 76），距终点 $Z=82$ 还差 6 → **6 次 β⁻ 衰变**。共 **8 α + 6 β⁻**（铀-镭系）。(§12.3)
16. **体内摄入（Inkorporation）**＝放射性物质**进入体内**（呼吸、食物、伤口）。**3 个 A** 失效，是因为**距离**（$r\to0$）和**屏蔽**都没了，**时间**也无法自由选择。体内的 **α 放射体**特别危险，因为它们在极短距离上把全部能量**直接释放进组织**（例：吸入氡、I-131 进甲状腺）。(§12.6)
