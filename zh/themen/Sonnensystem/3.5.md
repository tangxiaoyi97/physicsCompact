# 3.5 伽利略卫星

> **来源：** [[Themenpool_Physik_Matura_2026#3) Sonnensystem|考纲 §3]] · 课堂笔记 §8.2

## Io（木卫一）

**1610 年由伽利略发现**，以**朱庇特（木星神）的情人**命名；直径 **3643 km**，是太阳系**第四大**卫星。核为铁/镍/硫化铁，幔为硅酸盐岩。==**没有陨石坑 → 表面年轻**==：火山活动不断把它填平。

==**潮汐力驱动的火山活动**==：与木星及其他卫星的引力相互作用**反复挤压拉伸** Io → 内部摩擦 → 熔融 → 火山、液态硫、熔岩流。表面起伏可达 100 m。木星强磁场把周围的宇宙射线汇聚——Io 因此强烈受其照射。

> ✎ **什么是潮汐力——Io 都被锁定了，为什么还会被「揉捏」？** **潮汐力** = [[zh/themen/Gravitationsphysik/|引力]]在天体**近侧与远侧的差**（引力随距离衰减：朝向木星的一面被拉得更强）→ 天体沿连线方向被**拉长**。==这种形变叫**液滴效应（Tröpfcheneffekt）**：天体像一滴下落的水珠一样被拉长，呈现出**略带水滴/卵形**的形状==（拉长方向指向施加引力的天体）。地球上的海洋因此鼓起**两个**潮包：朝月一侧被月球引力多拉，背月一侧被绕共同质心的转动「甩」出去——所以每天**两次**涨潮（Epstein 谜题 181）。潮汐摩擦的长期后果是**潮汐锁定**：月球早已永远以同一面朝向地球——站在月球近侧永远看不到「地落」（谜题 163）；地球的自转也在被同样的机制缓缓刹车（谜题 164）。
>
> **刁钻追问：** Io 同样已被木星潮汐锁定——若轨道是正圆，形变就固定不变、**不再生热**。但 **Europa 和 Ganymed 的轨道共振**迫使 Io 的轨道保持**椭圆**：离木星时近时远，潮汐形变随之**周期性变化**——正是这持续的「揉捏」产生摩擦热。这就是课堂上说「与木星**及其他卫星**的相互作用」的深层含义。

## Europa（木卫二）

直径 **3122 km**，是**第六大**卫星。核为铁/镍，岩石幔，**表面为冰**。==其下是**液态咸水海洋**==，保持液态靠：

**潮汐力**（挤压拉伸产热）、**高盐度**（冰点降低）、**压强**（由于水的反常膨胀，液态水比冰更致密，受压时更长久地保持液态）。

冰壳有间歇泉式喷水，海底有火山活动。==**生命候选地**==——可设想冰面着陆器 + 海洋潜艇的探测（参见[[zh/themen/Astronomie und Exobiologie/1.6#1.6 其他行星上存在生命的论据|生命论据]]）。

## Ganymed（木卫三）

==**木卫三（Ganymed）**直径约 **5268 km**，是**整个太阳系最大的卫星**==——甚至比水星还大。三个特点：

1. ==**唯一拥有自身磁场的卫星**：== 没有别的卫星能产生自己的[[zh/themen/Elektrodynamik/|磁场]]——它源自一个液态、导电的**铁核**（发电机效应，与地球相同）。
2. **可能在冰层之间存在地下咸水海洋**（比 Europa 埋得更深、冰层更厚）→ 同样是潜在的生命之地。
3. 分层结构：铁核、岩石幔、厚厚的冰壳。

## Callisto（木卫四）

**木卫四（Callisto）**是四颗伽利略卫星中**最外侧**的一颗，几乎和水星一样大。它的标志恰好与 Io **相反**：

- ==**太阳系陨石坑最多、最古老的表面**：== Callisto 表面布满了密密麻麻的撞击坑。
- ==**地质上「死寂」**：== 因为 Callisto 运行在**更外侧**，[[#3.5 伽利略卫星|潮汐力]]几乎不再起作用 → **没有**火山活动、**没有**能抹去陨石坑的板块运动。远古的表面得以保留——它是早期撞击的一份「冻结」档案。

> ✎ **Io ↔ Callisto——同一原理，两个极端：** 二者都由同一机制[[#3.5 伽利略卫星|潮汐力]]塑造。**内侧**的 Io 被揉捏得极烈，成为**火山活动最剧烈**、表面**最年轻、无陨石坑**的天体；**外侧**的 Callisto 几乎感受不到潮汐，因而拥有**最古老、陨石坑最密**的表面。由内向外（Io → Europa → Ganymed → Callisto）的顺序，同时也是潮汐加热**递减**的顺序。
