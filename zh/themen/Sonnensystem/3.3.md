# 3.3 质子-质子链

> **来源：** [[Themenpool_Physik_Matura_2026#3) Sonnensystem|考纲 §3]] · 课堂笔记 §8.1.1

太阳最主要的聚变方式——四个氢核分步聚变成一个氦核。

![[assets/image79.png]]
*图：质子-质子链。第 1 步：两个质子 → 氘 + 正电子 + 中微子（β⁺）。第 2 步：氘 + 质子 → ³He + γ。第 3 步：两个 ³He → ⁴He + 两个质子。红=质子，蓝=中子。*

**第 1 步**——两个质子被压到一起。尽管有电斥力（[[zh/themen/Elektrodynamik/|库仑力]]），它们仍进入**强力**（[[zh/themen/Kernphysik/|核物理]]）的作用范围；一次 **β⁺ 衰变**把一个质子变成中子：

$$p^+ + p^+ \rightarrow D + e^+ + \nu_e$$

生成**氘**（重氢）。正电子立即与电子湮灭（粒子与反粒子相遇时彼此湮没，把全部质量转化为辐射）；中微子逃逸。==这一步**很罕见**==——大多数质子只是又弹开。

> ✎ **为什么第 1 步如此罕见——聚变又为何需要上千万度？** 两个质子**都带正电**、彼此排斥（[[zh/themen/Elektrodynamik/|库仑力]]随距离变小而**急剧**增大——即「库仑势垒」）。而能把它们束缚住的**强核力**，**只在极短距离**内起作用。要让两个质子靠得足够近，它们必须**非常快** → 这就需要核心那种极高温度（约 1500 万 °C）。此外还得在同一瞬间，由[[zh/themen/Kernphysik/|β⁺ 衰变]]（弱相互作用）把一个质子变成中子——这本身就是个小概率事件。两者叠加，使第 1 步罕见到——太阳**正因如此**才烧得这么慢、能持续数十亿年。

**第 2 步**——氘再俘获一个质子，首次聚变成**氦-3**并放出伽马射线：

$$D + p^+ \rightarrow {}^{3}\text{He} + \gamma$$

**第 3 步**——两个氦-3 核聚变成**氦-4**；两个质子被释放，可重新开启循环：

$${}^{3}\text{He} + {}^{3}\text{He} \rightarrow {}^{4}\text{He} + 2\,p^+$$

**净反应：**

$$4\,p^+ \;\rightarrow\; {}^{4}\text{He} + 2\,e^+ + 2\,\nu_e + 2\,\gamma + \text{能量}$$

**注意配平：** 第 3 步需要**两个** ³He，所以第 1、2 步每个循环各发生**两次**——净反应里才有 $2e^+$、$2\nu_e$、$2\gamma$。

> ✎ **释放多少能量——又究竟从哪来？** 关键是**质量亏损**：
> 1. 把**反应物**（四个质子）和**产物**（一个 ⁴He 核）分别称重。
> 2. ⁴He 比四个单独质子加起来**更轻**——「亏」掉了质量 $\Delta m$（约 **0.7%**）。
> 3. 这亏掉的质量并未消失，而是按 $E=mc^2$ 转化为**能量**：每个循环约 **26.7 MeV**（一小部分随中微子逃逸）。
>
> 这正是为什么**轻**核聚变会释放能量（见下文的结合能曲线）。(*[Wikipedia: Proton–proton chain](https://en.wikipedia.org/wiki/Proton%E2%80%93proton_chain)*)

> ✎ **两个经典追问（课本练习题，Mappe S. 30）：**
> - **为什么一杯水（满是氢核）里不会发生核聚变？** 室温下质子的动能比库仑势垒低了好几个数量级，而且质子被束缚在分子里、被电子云屏蔽——永远到不了「核的距离」（强力的作用范围）。聚变需要约 1500 万 °C 的等离子体。
> - **为什么每秒有数十亿个太阳中微子穿过我们，我们却毫无察觉？** 中微子不带电，只参与[[zh/themen/Elementarteilchenphysik/10.4#10.4 四种基本相互作用|弱相互作用]]——作用截面极小，整个地球对它们几乎是**透明**的。顺带一提：光子要「挤」**100 万年**才能离开太阳，中微子却**直接**从核心飞出（约 8 分钟后到达地球）——中微子探测器因此能「实时直播」太阳核心的聚变。

目前只到 **氢 → 氦**——太阳暂时没有足够的压强和温度去聚变氦。生命末期（红巨星阶段）可聚变到**碳**。==**到不了铁**==：只有质量更大的恒星才能聚变到铁。

> ✎ **为什么聚变偏偏止步于铁？（结合能曲线）** 逐步来看：
> 1. 每种核都有一个**每核子结合能**——它表示核被「捆」得有多紧、因而有多低能（多稳定）。这个值越大，核越稳定。
> 2. 把这个值对质量数作图，得到一条曲线，**峰值在铁**（$^{56}$Fe）——铁是**结合得最紧**、最稳定的核。
> 3. **峰值左侧**（轻核，H → He → … → Fe）：**聚变**时在曲线上**向上**移动 → 核子被捆得更紧，「多出来」的结合能按 $E=mc^2$ 以**质量亏损**释放出来。==所以轻核聚变会释放能量。==
> 4. **在峰顶（铁）：** 已经捆到最紧。再把铁聚变下去**不再**释放能量——相反，得**往里投**能量。
> 5. **峰值右侧**（比铁重）：这里只有**裂变（Fission）**才释放能量——所以人们让铀**裂变**而非聚变（[[zh/themen/Kernphysik/|核物理]]）。
>
> 因此铁是恒星聚变的天然「死胡同」：到铁之前恒星获得能量，从铁开始就要损失能量。
