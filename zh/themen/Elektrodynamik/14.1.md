# 14.1 电学简史

> **来源：** 课堂笔记 §5.1.1–§5.1.2

**古代**人们发现了**磁石**（在 Magnesia）和**琥珀**（希腊语 *elektron*）：琥珀摩擦皮毛会带电。==当时**磁与电**是两个分立现象==——尚不知其联系。

| 年份 / 人物 | 贡献 |
| :-- | :-- |
| ~1650 | **静电起电机**（一种靠摩擦/静电感应分离电荷、从而产生高压并打出火花的装置）——第一种（瞬时）电源 |
| 1752 **富兰克林** | 避雷针 |
| 1780 **伽伐尼（Galvani）** | 蛙腿抽动 → 电池的构想（两种金属 + 电解质） |
| 1800 **伏打（Volta）** | 第一个**电池**——首个稳定、均匀的电源 |
| 1820 **奥斯特（Ørsted）** | 电流产生**磁场**（指南针偏转） |
| 1831 **法拉第** | **电磁感应**（发电机原理） |
| 1864 **麦克斯韦** | 光是**电磁波** → 统一电磁学与光学（详见[[zh/themen/Optik und Wellenphaenomene/13.1#13.1 光的历史|主题 13]]） |
| 1888 **赫兹** | 产生并验证**电磁波** |
| 1901 **马可尼** | 首次跨大西洋**无线电** |
| 今天 | **量子电动力学（QED）**——电子 ↔ 光子相互作用 |

✎ 年代以课堂讲法为准，但各来源略有出入：麦克斯韦的统一各处记为 1856/1860/1864（其场论论文发表于 1864/65 年）；赫兹的实验跨 1886–88 年；严格说 ~1650 年格里克（Guericke）的硫磺球是**摩擦**起电机，狭义的 Influenzmaschine（静电**感应**起电机，如 Wimshurst 机）是 19 世纪的改良——口试沿用老师的统称即可。另注意：**海底电报电缆**自 1866 年起就已横跨大西洋——马可尼的突破恰恰在于**不用电缆**的无线传输。

> ✎ **交叉引用——闪电作为电学自然现象。** 自然界最壮观的电现象，**闪电与雷暴**（云中电荷分离、放电、雷声作为电磁/声学事件），物理上纯属电动力学，但本课程把它放在「自然现象」一章里讲：详见 **[[zh/themen/Physikalische Naturphaenomene/6.3#6.3 闪电与雷暴|主题 6 §6.3]]**（闪电的阶段、避雷针）与 **[[zh/themen/Physikalische Naturphaenomene/6.4#6.4 声强|§6.4]]**（雷声作为线声源）。防雷的机制正用到本主题的知识——见 **[[zh/themen/Elektrodynamik/14.11#14.11 法拉第笼|§14.11]]**（法拉第笼）与 **[[zh/themen/Elektrodynamik/14.12#14.12 跨步电压|§14.12]]**（跨步电压）。

> ✎ **伽伐尼与伏打之争——谁解释对了？** 伽伐尼（解剖学家）把蛙腿挂在金属架上，再用**另一种金属**的棒去碰：蛙腿抽动。他以为自己找到了**「生命的本质」**（「动物电」）。**伏打**不同意：真正的原因是==**两种不同金属 + 电解质**（蛙腿中导电的体液）构成了一个**电池**==，第二根金属棒一碰即**闭合电路** → 电流使肌肉收缩。伏打据此造出第一块电池——蛙腿在伽伐尼的装置里其实同时充当**电解质**和**电流指示器**。两人都有功：伽伐尼**发现**了现象，伏打**正确解释**并**利用**了它。

> ✎ **早期浪漫主义（Frühromantik）与奥斯特——为什么会有人去找电与磁的联系？（老师专门强调）** 早期浪漫主义是对启蒙运动**机械论、条块分割**式思维的反动；其基本思想：==**万物彼此相连**==（人、动物、植物、石头、天体……）。因此自然哲学家们猜想：**电现象与磁现象之间也应有联系**。奥斯特 1820 年的发现（电流使指南针偏转）正**证实**了这一猜想——一个哲学世界观引导出物理发现的著名例子。
